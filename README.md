# Currency Converter (HTML, CSS, JavaScript)

A responsive currency converter web application built using HTML, CSS, and JavaScript.  
It fetches real-time exchange rates from a public currency API and dynamically updates currency values.

---

## Features

- Real-time currency conversion using an external API
- Supports multiple international currencies
- Dynamic country flag updates based on selected currency
- Responsive and clean user interface
- Input validation for currency amount
- Modern UI with hover and interaction effects

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Font Awesome (icons)
- Flags API (country flags)

## How It Works

1. User enters an amount
2. Selects source and target currencies
3. Application fetches latest exchange rates
4. Converted value is displayed instantly
